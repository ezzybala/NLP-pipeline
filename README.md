# NLP-pipeline

An NLP pipeline is constructed from the following steps:

Data acquisition,
Text extraction and cleaning,
Pre-processing,
Feature Engineering,
Modelling,
Evaluation,
Deployement,
Monitoring & Model updating,

Data Acquisition
